# ALWebsocketLib
This is a Mod for Satisfactory(Game) that load the Websocket feature in background.
Also this mod using [IXWebsocket repo](https://github.com/machinezone/IXWebSocket?tab=readme-ov-file) to enable Websocket feature.

Read ALWebsocketBPObj.h for what you can do in BP.

Basically call "CreateWebSocket" to make Websocket Manager, then bind events and call functions. (Search Category: use "ALWebsocket")

Current Mod Version : 1.0.0
